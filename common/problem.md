# 错误排查指南

当遇到错误或Bug时，为了快速定位和解决问题，请按照以下格式提供详细信息：

## 问题排查模板

### 基本信息
- **调用方法名**：`[具体方法名称]`
- **输入参数**：`[参数详情]`
- **错误回调信息**：`[错误详情]`
- **SDK日志**：`[日志文件]`

### 问题描述
请详细描述：
1. 错误发生的具体场景
2. 期望的行为
3. 实际发生的行为
4. 错误信息或异常堆栈

### 环境信息
- **SDK版本**：`[版本号]`
- **运行环境**：`[操作系统/平台]`
- **相关依赖**：`[依赖版本]`

---

**注意**：提供的信息越详细，问题解决的速度越快。建议包含完整的错误日志和复现步骤。
